A fantasy football static website for viewing team rosters, league history, and player statistics. Integrated with the Sleeper's REST API.

## Features
- Consumes Sleeper API to display league, team and player data
- Dynamic front-end built with Javascript
- Responsive UI styled in modern CSS

## Tech Stack
- Javascript (ES6+)
- HTML5
- CSS3
  
## Demo
Live: [https://dgatanis.github.io/PublicFFL/](https://dgatanis.github.io/PublicFFL/)

## What I learned
- Fetching and handling external API data using JavaScript (async/await)
- Managing state and updating the DOM dynamically
- Building responsive layouts and animations with CSS
